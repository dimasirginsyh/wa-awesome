
# Whatsapp Awesome

Kamu penasaran apa yang dia kirim kepadamu menggunakan pesan sekali lihat?


## Installation

```bash
  cd wa-awesome
  npm Install
  npm start
```

Scan QR dengan whatsapp kamu. Jika console mengeluarkan kalimat **Client is ready!** aplikasi kamu sudah berjalan dan bisa mensalin pesan sekali lihat yang dikirim ke nomor kamu.
